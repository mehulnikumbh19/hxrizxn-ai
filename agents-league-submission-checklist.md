# Agents League Hackathon — Submission Checklist

**Deadline: 11:59 PM PT, June 14, 2026** (3 days from June 12, 2026)
**Platform: Innovation Studios Platform**

---

## ⚠️ First Step: Activate Your Profile
- An **active profile on the Innovation Studios Platform is required** to participate and submit.
- Activate via the link in the official email ("Activate your profile").
- **All team members** must individually register AND activate their profiles.

---

## Challenge Tracks
Submit to one track or across all three:

| Track | Description | Required Tool |
|-------|-------------|---------------|
| **Creative Apps** | Innovative applications with AI-assisted development | GitHub Copilot |
| **Reasoning Agents** | Intelligent agents solving complex problems via multi-step reasoning | Microsoft Foundry |
| **Enterprise Agents** | Business-ready agents | Microsoft 365 Copilot |

---

## What to Submit (Required Components)

1. **Working project** — built with the required developer tools, meeting the core requirements.
2. **Project description** — explains:
   - Features & functionality
   - Problem solved
   - Technologies used
3. **Demo video** — 2 minutes max
   - Show the project in action
   - Upload to YouTube or Vimeo; provide the **public link**
   - No third-party trademarks or copyrighted material (unless you have permission)
   - Add the video URL to the project description
4. **Public GitHub repository** — source code for judging/testing
   - Put the repo link in the project's **"Code Repository Link"** field
5. **Architecture diagram** — illustrating how the solution uses Microsoft Foundry, Agent Framework, Azure MCP, GitHub Copilot, and/or Azure services.

---

## Team Rules
- Teams: **max 5 members**.
- All members must register + activate profiles on Innovation Studios.
- Members must **request to be added to the Project** and be **accepted by the project originator** within the Innovation Studios tool.

---

## Final Pre-Submission Checklist
- [ ] Profile activated (you + all team members)
- [ ] Project created on Innovation Studios Platform
- [ ] Project description clearly explains the problem, solution, and AI value
- [ ] Video, code, demos, screenshots, and/or diagrams included
- [ ] Architecture diagram included
- [ ] Demo video (≤2 min) uploaded to YouTube/Vimeo, public link added to description
- [ ] Public GitHub repo link added to "Code Repository Link" field
- [ ] Setup / usage / walkthrough instructions are clear and easy to follow
- [ ] All team members added to and accepted into the project
- [ ] **Submission finalized before the hackathon closes (June 14, 11:59 PM PT)**
