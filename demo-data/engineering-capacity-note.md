# Engineering Capacity and Roadmap Note

Custom feature work for a single customer pulls engineering capacity off the core roadmap. A common outcome is a multi-month (often 6-month) delay to planned work.

Protect the team:
- Make prioritization explicit and visible.
- Set a realistic capacity ceiling before committing delivery dates.
- Watch for burnout and morale loss as early signals of over-commitment.
- Assume the delay will be longer than the optimistic estimate.
