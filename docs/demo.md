# Five-Minute Demo Script

## 0:00-0:30 Hook

Most AI tools give advice. Hxrizxn AI lets you test-drive plausible futures before you commit. The product is for high-stakes life choices where regret, reversibility, and second-order effects matter.

## 0:30-1:10 Canonical Decision

Enter: "I'm a software engineer with 3 years of experience. I have savings for 8 months. I want to quit my job and start an AI startup, but I'm worried about burn rate, isolation, and whether I'm romanticizing founder life. Should I quit now, wait 6 months, or test the idea part-time first?"

Mention that the supporting docs are mock grounding artifacts ready to be indexed through Foundry IQ: current budget, startup note, resignation pros/cons, and personal priorities.

## 1:10-2:00 Agents Running

Show the loading screen and agent trace:

- Decision Framing Agent
- Scenario Lattice Agent
- Ripple Effects Agent
- Risk and Black Swan Agent
- Optionality and Reversibility Agent
- Regret and Future Self Agent
- Experiment Design Agent
- Verifier and Safety Agent
- Recommendation Composer Agent

Say: every handoff is structured JSON, and traces are visible without exposing chain-of-thought.

## 2:00-3:00 Scenario Lattice

Show the three futures:

- Controlled Upside Branch
- Evidence-Building Base Case
- Runway Compression Stress Case

Point out reversibility, optionality, regret, upside, and downside score bars.

## 3:00-4:00 Ripple and Safety

Open the risk heatmap and impact chart. Highlight second-order consequences, runway compression, support-system fatigue, and black-swan external shock. Mention that high-stakes domains add disclaimers and professional-consultation guidance.

## 4:00-4:45 Experiment Plan

Show the 30-day reversible HORIZON-X validation sprint. The recommendation is not "quit" or "do not quit"; it is a safer experiment that collapses the biggest uncertainty before an irreversible move.

## 4:45-5:00 Closing

Hxrizxn AI does not predict your life. It reduces irreversible regret by making the next step smaller, safer, and more evidence-grounded. Judges should remember it as a decision laboratory, not another chatbot.
